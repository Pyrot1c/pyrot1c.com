    • Xbox Powered Arcade Cabinet (XPAC)

    This was a project I took on years ago in my free time to create my very own Mame cabinet, I took the unique approach in using an original xbox to host all the games and emulators instead of the traditional pc route. The system Runs a modified xbox dashboard accomplished using a software modified original xbox with a custom dash written to function with the Arcade controls. The emulators on the system include, Mame, Neogeo, Snes, Nes, N64, and Sega genesis with full romsets for each. The console also hosts xbox games which are native to the console This is all stored on a 500gb Western Digital HDD inside the xbox.
